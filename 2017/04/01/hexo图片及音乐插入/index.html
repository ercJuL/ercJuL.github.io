hexo图片及音乐插入

2017-04-01 11:15

图片插入

在根目录下的_config.yml中确认 post_asset_folder: true
用 hexo new file_name 时会自动创建同名文件夹
在file_name.md用 ![图片描述](图片文件全名) 可以直接使用同名文件夹下的图片

音乐插入

  1. 同图片放文件夹方式使用audio标签来使用

    以下属性仅支持html5游览器可用
    controls 显示播放控件，无控件参考文献 谷歌 学术 音乐
    autoplay 自动播放，大部分手机游览器会屏蔽此属性
    loop 循环播放
