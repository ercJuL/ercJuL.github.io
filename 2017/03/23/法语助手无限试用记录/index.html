使用软件

软件本身未加密，也为发现自检或暗装

火绒剑

过程

  1. 打开软件发现未注册且有试用次数限制，试用版可以使用全部功能
  2. 打开火绒剑监控–只包含frhelper.exe，发现可以注册表项

    3.转到注册表，显示0x31(49)，就是目标
  3. 编辑注册表导入，*.reg

    期间测试，发现首位不能为1，判断次数的变量应该是有符号数，最大可以为0x7fffffff
    5.重新打开软件

结束语

本身不难，软件未设暗装(暂未发现)，仅检测注册表项
